# 🩺 AI-Based Disease Prediction System

An AI/ML-based web application that predicts diseases based on user-selected symptoms.  
Built using Machine Learning and deployed using Streamlit.

## 📂 Project Structure

```
disease-prediction-app/
│── app.py              # Streamlit Web App
│── train_model.py      # Model Training Script
│── model.pkl           # Saved ML Model
│── disease.csv         # Dataset
│── requirements.txt    # Required Libraries
```

## 🧠 Machine Learning Model
- Algorithm Used: Logistic Regression
- Type: Classification
- Input: Symptoms
- Output: Predicted Disease

## ⚙️ How to Run Locally

```bash
git clone https://github.com/your-username/disease-prediction-app.git
cd disease-prediction-app
pip install -r requirements.txt
streamlit run app.py
```
